# EasyLife WoW Addon

EasyLife is a modular addon suite designed for World of Warcraft Classic Era (1.15.x). It enhances the gameplay experience by providing various utility modules that can be enabled or disabled based on player preferences.

## Features

- **Core Functionality**: The core addon provides a global `EasyLife` table, module registry, localization support, and a minimap button for easy access.
- **Modular Design**: Satellite addons can be added to extend functionality without cluttering the core experience. Each module can be independently enabled or disabled.
- **User-Friendly Configuration**: A dedicated configuration UI allows players to customize settings for each module easily.
- **Localization Support**: The addon supports multiple languages, making it accessible to a wider audience.

## Included Modules

- **Advertiser**: Helps players advertise their services in the game.
- **Boostilator**: Provides boosts and enhancements for gameplay.
- **VendorTracker**: Tracks vendor-related activities and items.
- **IceBlockHelper**: Assists players using the Ice Block ability.
- **AggroAlert**: Alerts players about aggro-related events.
- **RangeIndicator**: Displays range indicators for spells and abilities.
- **CastBarAura**: Provides functionalities related to cast bars.

## Installation

1. Download the latest release from the [releases page](#).
2. Extract the contents to your World of Warcraft `Interface/AddOns` directory.
3. Launch the game and enable the EasyLife addon from the AddOns menu.

## Usage

- Access the configuration UI through the minimap button or the main menu.
- Customize settings for each module according to your preferences.
- Enjoy enhanced gameplay with the various utilities provided by EasyLife.
